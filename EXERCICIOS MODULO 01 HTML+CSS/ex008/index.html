Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B que é uma tag não semantica, que nao tem significado tem forma

Tambem temos um termo em negrito usando a tag STRONG que ela é semantica alem de ela ter forma ela tem sentido

Itálico / Ênfase

Nesta frase temos um termo em italico usando a tag I nao sematica

E nesta frases temos um termo italico usando a tag EM que é semantica

Texto Marcado

Par marcamos um texto usando a tag MARK

Texto Grande e Pequeno

Estamos criando um texto grande Usamos a tag BIG e um texto pequeno neste paragrafo Usamos a tag SMALL

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas nao considerado Usamos a tag DEL

Texto Inserido

podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado Usamos a tag INS

Texto Subrescrito

Texto Subscrito

para inserir coisas do tipo H2O Usamos a tag SUB
